小喵喵RPG模組文檔

本頁面說明小喵喵RPG模組JSON結構、必填與可選欄位，以及生物/方塊生成規則。

必填欄位(type)

{
    "info": {
        "author": "Example",
        "modename": "example",
        "modeversion": "1.0"
    },
    "data": {
        "animal": {},
        "block": {},
        "item": {}
    },
    "rule":{
        "animal": {},
        "block": {}
    },
    "type": []
}
欄位型態說明
info.authorstr模組作者名稱
info.modenamestr模組名稱
info.modeversionstr模組版本號
data.animaldict生物資料字典
data.blockdict方塊資料字典
data.itemdict道具資料字典
rule.animaldict生物生成規則字典
rule.blockdict方塊生成規則字典
typelist模組類型字串列表(預設值default)

生物物品註冊

data.animal - 額外生物


"<生物名稱>": {
  "hp": 10,
  "drops": ["<掉落物>"],
  "display-emoji": "<emoji>"
}

data.block - 額外方塊


"<方塊名稱>": {
  "drops": ["<掉落物>"],
  "display-emoji": "<emoji>"
}

data.item - 額外物品

"<物品名稱>": {}
欄位型態說明
<生物名稱>.hpint生物血量
<生物名稱>.dropslist掉落物
<生物名稱>.display-emojistr顯示的表情符號
<方塊名稱>.dropslist掉落物
<方塊名稱>.display-emojistr顯示的表情符號
<物品名稱>dict尚無屬性

生物方塊生成規則

rule.animal


"<生物名稱>": {
    "chunk": {
        "max":<區塊最多生成數量>,
        "min":<區塊最少生成數量>
    },
    "world": {
        "max":<世界最多生成數量>,
        "min":<世界最少生成數量>
    },
  "type": ["<type>"]
}

rule.block


"<方塊名稱>": {
    "chunk": {
        "max":<區塊最多生成數量>,
        "min":<區塊最少生成數量>
    },
    "world": {
        "max":<世界最多生成數量>,
        "min":<世界最少生成數量>
    },
    "type": ["<type>"]
}
欄位型態說明
<生物名稱>.chunkdict區塊生成規則
<生物名稱>.chunk.maxint區塊最多生成數量
<生物名稱>.chunk.minint區塊最少生成數量
<生物名稱>.worlddict世界生成規則
<生物名稱>.world.maxint世界最多生成數量
<生物名稱>.world.minint世界最少生成數量
<生物名稱>.typelist生物類別
<方塊名稱>.chunkdict區塊生成規則
<方塊名稱>.chunk.maxint區塊最多生成數量
<方塊名稱>.chunk.minint區塊最少生成數量
<方塊名稱>.worlddict世界生成規則
<方塊名稱>.world.maxint世界最多生成數量
<方塊名稱>.world.minint世界最少生成數量
<方塊名稱>.typelist方塊類別

驗證規則

  • 必填欄位必須存在，型態符合必填欄位
  • 可選欄位若存在，型態必須符合生物物品註冊
  • data 中的生物、方塊必須對應 rule 中的生成規則
  • rule 中的生物、方塊必須在 data 中有資料。
  • 若缺少 hp、drops、display-emoji 等關鍵欄位就會報錯。

完整範例

{
    "info": {
        "author": "command_cat",
        "modename": "example",
        "modeversion": "v1.2"
    },
    "data": {
        "animal": {
            "cat": {
                "hp":10,
                "drops":["bone"],
                "display-emoji":"🐱"
            }
        },
        "block": {
            "stone": {
                "drops":["stone"],
                "display-emoji":"🟫"
            }
        },
        "item": {
            "bone": {}
        },
        "map": [] // 可暫時忽略為空
    },
    "rule": {
        "animal": {
            "cat": {
                "chunk":{
                    "max":3,
                    "min":1
                },
                "world":{
                    "max":100,
                    "min":50
                },
                "type":["default", "normal"] // 可暫時忽略為空
            }
        },
        "block": {
            "stone": {
                "chunk":{
                    "max":5,
                    "min":3
                },
                "world":{
                    "max":200,
                    "min":100
                },
                "type":["default"] // 可暫時忽略為空
            }
        }
    },
    "type": ["normal"]
}
